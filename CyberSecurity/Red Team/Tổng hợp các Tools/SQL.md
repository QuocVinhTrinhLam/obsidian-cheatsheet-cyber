| **Nhóm lệnh**        | **Mục đích**              |
| -------------------- | ------------------------- |
| SELECT               | Lấy dữ liệu               |
| WHERE                | Lọc điều kiện             |
| ORDER BY             | Sắp xếp                   |
| LIMIT                | Giới hạn kết quả          |
| DISTINCT             | Loại trùng                |
| GROUP BY             | Gom nhóm                  |
| HAVING               | Lọc sau khi group         |
| JOIN                 | Kết hợp bảng              |
| CONCAT               | Nối chuỗi                 |
| GROUP_CONCAT         | Gom + nối chuỗi sau group |
| AS                   | Đặt tên tạm               |
| INSERT/UPDATE/DELETE | Thêm/Sửa/Xoá nếu có học   |
